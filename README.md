# Web服务器

**简介：**利用WinSock编程接口编写简单的Web服务器程序，可以接收IE等Web浏览器的请求，对HTTP请求进行处理，向浏览器客户发送HTTP响应，由浏览器显示请求的页面，win8.1实测可用。

**开发环境：**VS2015

**开发语言：**C++

**用法：**

* www文件夹下存放html页面
* 先双击打开WebServer.exe
* 打开浏览器输入127.0.0.1:81//baidu.html(如果输入其他页面会显示NotFound页面)
